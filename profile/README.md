# XMRT DAO

**Open-Source AI Infrastructure for Privacy, Health & Decentralized Compute**

XMRT DAO builds production-grade AI systems that run natively on AMD hardware — from vision therapy to GPU kernel optimization to zero-knowledge governance. Our work spans healthcare, Web3, multimodal AI, and developer tooling.

All projects are **MIT licensed**, built in public, and deployed from a phone via Termux + Ollama.

## Stack

- **Compute:** AMD Instinct MI300X, ROCm 6.2
- **Inference:** ONNX Runtime MIOpen EP, vLLM ROCm
- **Training:** PEFT QLoRA, TRL GRPO
- **Models:** Qwen2.5-Coder, Qwen2.5-VL, YOLOv8, MediaPipe, Piper
- **Blockchain:** Monero (XMR), Solidity DAO contracts
- **DevOps:** GitHub Actions → Hugging Face Spaces auto-deploy
